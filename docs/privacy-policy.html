Privacy Policy

Last updated: January 2026

1. Introduction

Smart Row Highlighter ("we", "our", or "the App") is a monday.com marketplace application that helps users visually highlight rows based on conditional rules. This Privacy Policy explains how we collect, use, and protect your information.

2. Information We Collect

Smart Row Highlighter accesses the following data through the monday.com API:

  • Board Data: Item names, column values, and column configurations from boards where you use the app
  • Storage Data: Your highlighting rules are stored using monday.com's secure Storage API

We do NOT collect, store, or transmit any of this data to external servers. All data processing occurs entirely within the monday.com platform.

3. How We Use Your Information

The data accessed is used solely to:

  • Display items and columns in the app interface
  • Evaluate highlighting rules against your board data
  • Save and retrieve your highlighting rules using monday.com Storage API
  • Apply visual highlighting to rows that match your configured rules

4. Data Storage

Smart Row Highlighter stores your highlighting rules using monday.com's built-in Storage API. The app:

  • Does not have external databases
  • Does not transmit data to third-party services
  • Stores rules securely within monday.com's infrastructure
  • Processes all data in real-time through monday.com's secure API

5. Data Security

Your data security is ensured through:

  • monday.com's OAuth 2.0 authentication
  • monday.com's secure API infrastructure
  • monday.com's Storage API encryption
  • No external data transmission or storage

6. Third-Party Services

Smart Row Highlighter does not integrate with or share data with any third-party services. The app operates exclusively within the monday.com ecosystem.

7. Your Rights

You can:

  • Export your highlighting rules at any time using the Export feature
  • Delete all your rules by removing them from the app
  • Uninstall the app at any time from your monday.com account
  • Revoke API permissions through monday.com's app management
  • Contact us for any privacy-related questions

8. Children's Privacy

Smart Row Highlighter is a business productivity tool and is not intended for children under 13. We do not knowingly collect information from children.

9. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes through the monday.com marketplace listing.
