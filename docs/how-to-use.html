Smart Row Highlighter

Automatically highlight rows based on column values with powerful conditional rules

1 Getting Started

Smart Row Highlighter is a Board View that lets you create visual highlighting rules for your monday.com boards. After installation, add it to any board:

  1. Open your board
  2. Click the + Add View button
  3. Search for "Smart Row Highlighter"
  4. Click to add it to your board

You'll see your board items in a preview table and an empty rules panel. Let's create your first rule!

2 Creating Your First Rule

Click the "+ Add Rule" button to open the rule editor:

  1. Rule Name: Give your rule a descriptive name (e.g., "Overdue Tasks")
  2. Column: Select which column to evaluate
  3. Operator: Choose a condition (equals, contains, is overdue, etc.)
  4. Value: Enter the value to match (if required)
  5. Highlight Color: Pick a color for matching rows
  6. Click "Create Rule" to save
Pro Tip
Operators like "is overdue", "is today", and "is empty" don't require a value - they evaluate automatically based on the current date or column state.

4 Multiple Conditions (AND/OR)

Create powerful rules by combining multiple conditions:

  1. Click "+ Add Condition" in the rule editor
  2. Configure each condition independently
  3. Choose AND (all must match) or OR (any must match)

Examples:

  • AND: Status = "In Progress" AND Due Date is overdue ‚Üí Highlight urgent tasks
  • OR: Priority = "High" OR Priority = "Critical" ‚Üí Highlight all high-priority items

5 Managing Your Rules

Rule Priority (Drag & Drop)

When multiple rules could match the same row, the first matching rule wins. Drag rules to reorder them and set priorities.

Available Actions

  • Toggle On/Off: Temporarily disable a rule without deleting it
  • Edit: Click the settings icon to modify a rule
  • Duplicate: Create a copy of an existing rule
  • Delete: Remove a rule permanently

Search Rules

When you have more than 3 rules, a search box appears to quickly find rules by name.

6 Export & Import Rules

Share your highlighting configuration across boards or with team members:

Export

  1. Click the Export button
  2. A JSON file will download with all your rules

Import

  1. Click the Import button
  2. Paste the JSON content from an exported file
  3. Click "Import Rules" to add them
Note
Imported rules are added to your existing rules. Make sure column names match between boards for rules to work correctly.

7 Available Highlight Colors

Choose from 8 carefully selected colors that work well in both light and dark modes:

  • Red - Urgent, overdue, critical
  • Pink - Special attention
  • Purple - VIP, premium
  • Indigo - In review
  • Blue - In progress
  • Green - Completed, approved
  • Yellow - Warning, pending
  • Orange - Medium priority

8 Board Preview

The Board Preview shows how your rules affect your items in real-time:

  • See highlighted rows as you create and modify rules
  • Hover over an item name to see which rule(s) matched
  • Multiple matching rules show a badge with the count
  • Pagination available for boards with many items (20 per page)
