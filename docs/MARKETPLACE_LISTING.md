## Long Description (200-2,500 characters)

```
Smart Row Highlighter automatically highlights rows in your monday.com boards based on conditional rules you define. Make important items stand out instantly!

KEY FEATURES:

• Rule-Based Highlighting - Create rules that evaluate column values and apply colors to matching rows

• Multiple Conditions - Combine conditions with AND/OR logic for powerful filtering (e.g., "Status = Done AND Due Date is overdue")

• 15+ Operators - equals, contains, starts with, ends with, regex, greater than, less than, is overdue, is today, is this week, and more

• All Column Types - Works with Status, Text, Numbers, Date, Checkbox, and People columns

• Drag & Drop Priority - Reorder rules to control which highlighting takes precedence

• Export & Import - Share your rule configurations across boards or with team members

• Real-Time Preview - See highlighting applied instantly as you create and modify rules

• Dark Mode Support - Looks great in both light and dark themes

PERFECT FOR:

- Project managers tracking overdue or at-risk tasks
- Sales teams highlighting high-value opportunities
- HR teams flagging pending approvals
- Support teams identifying urgent tickets
- Anyone who wants better visual organization

Free users get 5 powerful rules per board. With multiple conditions per rule, that's more than enough for most workflows!
```
(1,197 characters)
